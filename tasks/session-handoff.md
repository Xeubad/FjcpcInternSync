> 下一次 Claude Code 会话开工的第一件事是读这份。

## 维护原则

- 最新交接放最顶部
- 每条交接必须有：本轮主线 / 状态 / 已完成 / 进行中 / 阻塞 / 下次第一动作 / 风险
- 历史交接保留最近 3 条作为短期记忆；更老的合并归档到 progress.md
- "下次会话建议第一动作"必须是**具体可执行的动词**，不能是"建议关注一下..."
- 短任务（一次性问答 / 无文件改动）可以不更新本文件，但要在最终回复里明示"本轮无需落盘"
